BSScanner
====

## Overview
---

Bluetooth Smart(Bluetooth Low Energy)及び無線LAN，環境音を用いた位置推定のテストのためのAndroidアプリ

## Install
---

Download the APK package from [github](https://github.com/YA-androidapp/BSScanner/blob/master/app/app-release.apk?raw=true)
